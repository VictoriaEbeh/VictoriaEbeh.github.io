Course Outline

Source: UTD Graduate Catalogue

GISC 6301: GIS Data Analysis Fundamentals

In this course, we explore essential statistical techniques and algorithms for handling geospatial data. Key topics include descriptive statistics, data visualization, exploratory methods, probability theory, study design, and sampling theory. We also cover statistical inference, simulation experiments, basic correlation and regression analyses, and pattern analysis, all from a Geoinformation Sciences perspective. This ensures an understanding of how to analyze and interpret geospatial data effectively.

GISC 6325: Remote Sensing Fundamentals

This course introduces the principles of remote sensing, sensor technologies, image processing techniques, and their applications. We learn about electromagnetic radiation, satellite and airborne systems, and how to use remote sensing data for real-world problems. Hands-on exercises used state-of-the-art software such as ERDAS Imagine.

GISC 6384: Advanced Geographic Information Systems

This course covers the basics and more advanced topics of Geographic Information Systems (GIS) and their applications. I learned how to use GIS for manipulating, querying, analyzing, and visualizing spatial data. Topics include raster and vector data models, Geodatabase, map algebra, 3-D surface analysis, spatial interpolation, and network analysis. Lab exercises with industry-standard GIS software provided hands-on experience solving real-world social, economic, and environmental issues.

GISC 6382: Applied Geographic Information Systems

This course teaches hands-on skills with industry-standard GIS software for diverse applications. We apply GIS techniques to manage urban infrastructure, analyze marketing and locations, address environmental issues, and explore geologic and geophysical data. The course also covers applications in Economic, Political, and Policy Sciences, giving you hands-on experience in solving real-world challenges.

GISC 6354: Information Management

This course covers the fundamentals of database systems, including their concepts, design, programming, and management. It offers hands-on training in DBMS and explores new technologies like NoSQL and various modern database systems. I was engaged in database programming and application development, using programming languages such as SQL, Python, HTML, and Dash.

GISC 6323: Machine Learning for Socio-Economic and Geo-Referenced Data

This course introduces models, algorithms, and their conceptual foundations for structuring dynamic socio-economic and geo-referenced data. We utilize open-source software and commonly available hardware. Through practical examples, I learned supervised machine learning to develop classification rules and unsupervised data mining to reveal hidden data patterns.
